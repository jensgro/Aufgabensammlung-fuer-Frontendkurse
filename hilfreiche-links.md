## Bilder responsive machen

Jedem Bild, das sich über wechselnde Breiten skalieren soll, sollte diese Klasse mitgegeben werden (der Klassenname ist natürlich individuell wählbar):

````
    .flexible {
        width: 100%;
        max-width: 100%;
        height: auto;
    }
````
